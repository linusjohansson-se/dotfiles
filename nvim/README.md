# Keybinds in General

| Action | Keybinding | Function |
| :--- | :--- | :--- |
| **Search Files** | `<leader><space>` | Find any file in the solution |
| **Search Text (Grep)** | `<leader>/` | Search for actual code strings |
| **Search Symbols** | `<leader>ss` | Find classes/methods in current file |
| **Search Workspace** | `<leader>sS` | Find classes/methods in whole solution |
| **Recent Files** | `<leader>fr` | Show your last opened files |
